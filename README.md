# 🪐 Zenith - Il tuo Vault Personale (Your Personal Vault)

*[English version below]*

> "Lavorare sui tuoi pensieri come se fossi dentro la tua stessa mente."

**Zenith** è un ecosistema digitale ibrido che fonde un blocco note avanzato con un calendario dinamico, progettato per essere un'estensione della tua memoria. 

---

## 🇮🇹 La Visione (Italiano)

### 🤖 Il Primo Progetto AI-Driven
Zenith rappresenta un esperimento ambizioso: **è un progetto creato interamente attraverso l'uso dell'Intelligenza Artificiale (anche questo messaggio è stato scritto con IA)**.
Volevo testare i confini della tecnologia attuale per capire fino a che punto un individuo può spingersi nella creazione di un software complesso (gestione file, crittografia, Progressive Web App, grafi relazionali) senza dover scrivere manualmente ogni singola riga di codice, ma agendo come un **Architetto di Prompt**.

### Gli Obiettivi:
Essendo la mia prima esperienza nel mondo dello sviluppo, Zenith nasce con tre obiettivi chiari:
1. **Apprendimento:** Capire l'architettura dei software moderni.
2. **Privacy:** Creare uno strumento *offline-first* dove i dati appartengono solo all'utente.
3. **Ergonomia Cognitiva:** Realizzare un'interfaccia che non sia solo uno strumento, ma uno spazio mentale fluido.

### ✨ Caratteristiche Principali
- 📝 **Note-taking Avanzato:** Scrittura fluida con supporto tag e organizzazione in cartelle.
- 📅 **Calendario Integrato:** Visualizza i tuoi impegni e le tue note in una timeline temporale.
- 🕸️ **Graph View:** Visualizza le connessioni tra i tuoi pensieri come una rete neurale.
- 🔒 **Sicurezza Locale:** Crittografia dei dati sensibili (AES) direttamente nel browser.
- 📱 **PWA & Offline:** Funziona senza internet, con installazione immediata su smartphone.

## 🇬🇧 The Vision (English)

> "Working on your thoughts as if you were inside your own mind."

**Zenith** is a hybrid digital ecosystem that merges an advanced notepad with a dynamic calendar, designed to be an extension of your memory.

### 🤖 The First AI-Driven Project
Zenith represents an ambitious experiment: **it is a project created entirely through the use of Artificial Intelligence (even this message is written using AI)**.
I wanted to push the boundaries of current technology to understand how far an individual can go in creating complex software (file management, cryptography, Progressive Web Apps, relational graphs) without manually writing every single line of code, acting instead as a **Prompt Architect**.

### The Goals:
Being my first experience in the development world, Zenith was born with three clear goals:
1. **Learning:** Understanding modern software architecture.
2. **Privacy:** Creating an *offline-first* tool where data belongs solely to the user.
3. **Cognitive Ergonomics:** Building an interface that isn't just a tool, but a fluid mental space.

### ✨ Main Features
- 📝 **Advanced Note-taking:** Fluid writing with tag support and folder organization.
- 📅 **Integrated Calendar:** Visualize your tasks and notes on a time-based timeline.
- 🕸️ **Graph View:** See the connections between your thoughts like a neural network.
- 🔒 **Local Security:** Sensitive data encryption (AES) directly in the browser.
- 📱 **PWA & Offline:** Works without the internet, with instant installation on smartphones.
